## Other Ways to Contribute

Not in a position to sponsor financially? You can still help!

- **Star the repository** - It helps with visibility
- **Report bugs** - Help us improve quality
- **Contribute code** - See [CONTRIBUTING.md](CONTRIBUTING.md)
- **Spread the word** - Tell other game developers about Bannou
- **Write tutorials** - Help others learn the platform
- **Answer questions** - Help out in Discord or GitHub Discussions
